### Setup

1- Clone the repository

```
    git clone https://github.com/AbdelrhmanAmin/StartFI-Task.git
```

2- Open the folder.

```
    cd StartFI-Task
```

3- install the app

```
    npm install
```

4- Run the app

```
    npm start
```

5- Test the app

```
    npm test
```
